<Update label="May 25, 2026">

## New features

- **Realtime speech-to-speech in the browser.** Test agents end-to-end from your browser with low-latency, full-duplex voice. Open an agent in the [browser test panel](/browser-test) to try it.
- **Agent identity pipeline controls.** Configure persona, voice, and pipeline behavior per agent from [Agent Studio](/agent-studio) and the [configuration page](/configuration).

## Updates

- **More reliable device voice runtime.** Improved stability for voice turns running on Dot devices, including the [device realtime bridge](/dot-device).
- **Bundled Wi-Fi provisioning portal.** The OpenDot Wi-Fi portal now ships with device firmware for a smoother pairing flow. See [device, firmware & hardware](/device-firmware-hardware).
- **Hardened release pipeline.** Credentials are redacted from release artifacts and contribution guidelines are refreshed. See [contributing](/contributing).

## Fixes

- Stabilized the device realtime bridge to reduce dropped sessions.
- The realtime test panel is now included in the platform build so it loads consistently in the console.

</Update>
